| Тип данных | Описание |
|------------|----------|
| `ExtensionModule-Props` | Параметры, передаваемые в модуль расширения через **connector**. Это расширенные свойства компонента. |
| ```java 
ExtensionModule-InternalProps 
``` | Параметры, необходимые для внутренней работы модуля расширения. Используется в библиотеке для внутренней логики. |
| `ExtensionModule-Meta` | Мета-данные модуля расширения. Используется в библиотеке для регистрации модуля. |
